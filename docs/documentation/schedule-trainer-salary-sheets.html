Листы ЗП

Файл: /docs/documentation/schedule-trainer-salary-sheets.html
Четвёртая вкладка раздела «Журнал расписания». Архив сформированных слепков ЗП (только просмотр). Создание слепков — на вкладке «ЗП тренеров» (schedule.trainerSalary.manage).

1) Права

Достаточно schedule.trainerSalary.view (то же, что для «ЗП тренеров»). schedule.view не обязателен.

СитуацияРезультат
Гостьредирект / JSON 401
Нет schedule.trainerSalary.view403
Есть viewсписок, data, карточки batch/snapshot → 200
Слепок другого партнёракарточка snapshot → 404

2) Маршруты

Группа can:schedule.trainerSalary.view в routes/web.php.

МетодURLИмяНазначение
GET/schedule/trainer-salary-sheetsschedule.trainer-salary-sheetsHTML: список листов + фильтры
GET/schedule/trainer-salary-sheets/dataschedule.trainer-salary-sheets.dataJSON: таблица листов, блок «актуальные по тренерам»
GET/schedule/trainer-salary-sheets/batch/{batchId}schedule.trainer-salary-sheets.batch.showПолный лист пакета «Сформировать всех» (readonly)
GET/schedule/trainer-salary-sheets/snapshot/{id}schedule.trainer-salary-sheets.snapshot.showЛист одного тренера (версия vN, readonly)

Параметры списка: year, month, latest_only=1 (только актуальные версии / последний пакет).

3) Типы листов

  • Полный лист — все слепки с одним batch_id (кнопка «Сформировать всех»). В списке — одна строка на пакет.
  • По тренеру — слепок без batch_id («Сформировать» в строке); в списке — отдельные строки с версией vN.

Колонка «Сумма итого» — сумма total по всем строкам листа. Ссылки «Открыть» ведут на batch или snapshot show.

4) UI

  • Список: trainer_salary_sheets.blade.php, JS trainer-salary-sheets.js.
  • Деталь: trainer_salary_sheet_show.blade.php, таблица _trainer_salary_sheet_detail_table.blade.php (класс trainer-salary-table--readonly).
  • Бейдж «Актуальная» — последняя version тренера за выбранный месяц.
  • Бейдж «Последний пакет» — последний полный batch_id за месяц.
  • Блок «Актуальная версия по каждому тренеру» — сводка последних слепков (при выключенном «Только актуальные»).

5) Бэкенд

  • Контроллер: ScheduleTrainerSalarySheetsController.
  • Сервис: TrainerSalarySheetService.
  • Запрос: TrainerSalarySheetsReportRequest.

6) Тесты

php artisan test --env=testing --filter=ScheduleTrainerSalarySheets

Общий фильтр ЗП: --filter=ScheduleTrainerSalary (включает и листы, и вкладку черновика).
