История изменений на страницах CRM

На перечисленных ниже экранах в toolbar есть кнопка «История» (иконка fa-clock-rotate-left) → модалка includes.logModal → DataTables GET …/logs-data. Запись — только через AuditLogger в my_logs (event + level); в модалке показываются события одной категории (AuditEvent::eventValuesForCategory()). Подробнее — audit-my-logs, UI-паттерн — reusable-ui-partials § logModal.
