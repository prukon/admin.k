Пользователи в админке

Файл: /docs/documentation/admin-users.html

Создание (POST /admin/users)

Валидация: App\Http\Requests\User\StoreRequest. Поле password необязательно. Если пароль не указан или передана пустая строка, в таблице users сохраняется password = NULL (для учётных записей без входа в систему, например только для внутреннего учёта). Если пароль указан, действуют ограничения min:8 / max:255; хеширование выполняется кастом модели User (password → hashed), при NULL хеш не считается.
