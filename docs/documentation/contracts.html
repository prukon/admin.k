Договоры (клиентские): скачивание в интерфейсе

Папка: /docs/documentation
Правило единое для карточки договора в CRM (/client-contracts/{id}) и для вкладки «Мои документы» в учётной записи пользователя.

Какая кнопка показывается

  • Если статус договора signed (Contract::STATUS_SIGNED) — в интерфейсе предлагается только скачивание подписанного файла. Подпись кнопки/ссылки: «Скачать подписанный».
  • Если статус любой другой — предлагается только скачивание оригинала (неподписанного PDF). Подпись: «Скачать оригинал».

Замечания

  • Критерий выбора кнопки — именно поле contracts.status, а не наличие signed_pdf_path.
  • Если статус signed, а файл подписанного PDF в хранилище отсутствует или путь в БД не совпадает с диском, скачивание вернёт ошибку (см. обработку в контроллерах скачивания).
  • Файлы хранятся под корнем диска local: storage/app/ + относительный путь из source_pdf_path / signed_pdf_path.
